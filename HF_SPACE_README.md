# Content Engine

Free-mode deployment of the Content Engine web app.

- Docker-based FastAPI site
- Free trend discovery from Reddit and Google News RSS
- Free script generation via local heuristics
- Free voice generation via `espeak-ng`
- FFmpeg rendering for vertical short-form videos

The app listens on port `8000`.
